Cara Order

>

Cara Order Untuk melakukan order di Betta Id cukup mudah, tanpa registrasi member dan tanpa metode tambahkan ke keranjang.
Silahkan ikuti langkah-langkah berikut:
1. Pilih ikan cupang hias yang anda inginkan.
2. Hubungi kami melalui Telp / SMS / WhatsApp / BBM / Line untuk mengecek ketersediaan ikan yang anda pilih dengan menyebutkan judul ikan dan kode ikan.
3. Jika ikan yang anda pilih tersedia, segera transfer dana sebesar harga ikan + ongkos kirim ke rekening kami.
4.Setelah mentransfer ke rekening kami, lakukan konfirmasi melalui SMS / WhatsApp / BBM / Line dengan menyertakan bukti transfer.
5. Konfirmasi alamat lengkap pengiriman (nama + alamat lengkap + kode pos + No. Telp / HP)
6. Ikan cupang hias anda akan segera kami kirimkan.

Informasi & Jenis Order Di Betta Id
1. Tanya jawab seputar Betta Id silahkan lihat halaman FAQ.
2. Harga yang tertera di web adalah harga eceran / satuan.
3. Order grosir minimal 6 ikan (bebas campur) mendapat potongan harga 20%.
4. Order partai minimal 100 ikan (khusus ikan partai) dengan harga sebagai berikut:
Giant @Rp. 25.000
Halfmoon @Rp. 7.000
Plakat @Rp. 7.000
